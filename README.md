# Financial Crime Detection

![Financial Crime Detection Tests](https://github.com/Djones-qa/financial-crime-detection/actions/workflows/crime-detection-tests.yml/badge.svg)

This repository contains tools and tests for generating simulated financial transaction data and detecting fraud patterns, including velocity abuse, structuring, and geographic anomalies.

## Setup & Testing

1. Install dependencies:
   ```bash
   pip install -r requirements.txt
   ```
2. Run pytest suite:
   ```bash
   python -m pytest tests/
   ```
